HTML - Wat is HTML?

Html is de basisbouwsteen van het web. Het is een eenvoudige opmaaktaal die wordt gebruikt om webpagina's te maken. Stel je HTML voor als de 'skeletstructuur' van een pagina. Het geeft instructies aan de webbrowser over hoe de inhoud van een pagina moet worden weergegeven. HTML is gemakkelijk te leren - u zult ervan genieten!

Eerste HTML code:

  • De !DOCTYPE html verklaring definieert dat dit document een HTML5- document is.
  • Het html element is het hoofdelement van een HTML-pagina.
  • Het head bevat meta-informatie over de HTML-pagina.
  • Het meta is voornamelijk voor het aangeven wat er in de website staat en waar het over gaat, om hoger in het zoek systeem te komen.
  • De charset="UTF-8" in de meta specificeert een karakterset.
  • Het title element specificeert een titel voor de HTML-pagina (die wordt weergegeven in de titelbalk van de browser of op het tabblad van de pagina).
  • Het bodyelement definieert de hoofdtekst van het document en is een container voor alle zichtbare inhoud, zoals koppen, alinea's, afbeeldingen, hyperlinks, tabellen, lijsten, enz.
  • Het h1 element definieert een grote kop.
  • Het p element definieert een paragraaf.

HTML - Comments

HTML Comment Tag:

U kunt opmerkingen aan uw HTML-bron toevoegen met behulp van de volgende syntaxis:
!-- Write your comments here --
Opmerkingen worden niet door de browser weergegeven, maar kunnen wel helpen bij het documenteren van uw HTML-broncode. Met opmerkingen kunt u meldingen en herinneringen in uw HTML-code plaatsen. Opmerkingen kunnen worden gebruikt om inhoud te verbergen. Dit kan handig zijn als u inhoud tijdelijk verbergt

HTML - Opzet van een pagina

Wat is HTML(HyperTextMarkup Language)?

Html is de basisbouwsteen van het web. Het is een eenvoudige opmaaktaal die wordt gebruikt om webpagina's te maken. Stel je HTML voor als de 'skeletstructuur' van een pagina. Het geeft instructies aan de webbrowser over hoe de inhoud van een pagina moet worden weergegeven. HTML is gemakkelijk te leren - u zult ervan genieten! Een eenvoudig HTML-document:

<!DOCTYPE html>
<html>

<head>
<title > Page Title </title>
</head>

<body>
<h1 > My First Heading </h1>
<p > My first paragraph. </p>
</body>

</html>


Voorbeeld uitgelegd:

  • De !DOCTYPE html definieert dat dit document een HTML5- document is.
  • Het html is het hoofdelement van een HTML-pagina.
  • Het head bevat meta-informatie over de HTML-pagina.
  • Het titleelement specificeert een titel voor de HTML-pagina (die wordt weergegeven in de titelbalk van de browser of op het tabblad van de pagina).
  • Het body definieert de hoofdtekst van het document en is een container voor alle zichtbare inhoud, zoals koppen, alinea's, afbeeldingen, hyperlinks, tabellen, lijsten, enz.
  • Het h1 definieert een grote kop.
  • Het p definieert een paragraaf.

LET OP: De eerste pagina die je maakt heet de index (inhoudsopgave). Daarom is de eerste pagina dus altijd index.html

p She is a girl /p In dit voorbeeld pen /p zijn de start en eind tags en “She is a girls” is de inhoud.

Alle HTML-elementen kunnen attributen hebben.Attributen bieden aanvullende informatie over elementen.

HTML headings

HTML headings zijn titels of ondertitels die u op een webpagina wilt weergeven. Ze worden gedefinieerd met de h1 tot h6

<h1> De belangrijkste kop. </h1>
<h6> De minst belangrijke kop. </h6>

HTML paragraphs

Het HTML - p element definieert een paragraaf.

Voorbeeld:

<p> Is a paragraph. </p>
<p> Is another paragraph. </p>

HTML Links - Hyperlinks

HTML-links zijn hyperlinks. U kunt op een link klikken en naar een ander document springen. De HTML-a definieert een hyperlink. Het heeft de volgende syntaxis: link text Het belangrijkste attribuut van het a element is het href attribuut, dat de bestemming van de link aangeeft. De linktekst is het gedeelte dat zichtbaar is voor de lezer. Als u op de linktekst klikt, wordt de lezer naar het opgegeven URL-adres gestuurd.

Voorbeeld:

Dit voorbeeld laat zien hoe u een link naar W3Schools.com maakt:

<a href="https://www.w3schools.com/"> Visit W3Schools.com! </a>

Tip!: als jij het attribuut target="_blank" aan deze link toevoegt dan de browser zal de link in een nieuwe pagina openen

<a href="https://www.w3schools.com/" target=”_blank”> Visit W3Schools.com! </a>

HTML images:

Om een afbeelding toe te voegen aan je HTML-document hoef je alleen maar een regel tussen te voegen met de tag img. Hiermee worden de beelden vastgelegd in HTML. Om een afbeelding op een pagina weer te geven, moet je het attribuut src en het attribuut alt gebruiken. Src= staat voor "source". Het specificeert het pad (URL) naar de afbeelding. Alt =specificeert een alternatieve tekst voor de afbeelding. Als een browser een afbeelding niet kan vinden, wordt de waarde van het alt attribuut weergegeven.

<img src="url" alt="alternatetext">

HTML lay-out elementen:

HTML heeft verschillende semantische elementen die de verschillende delen van een webpagina definiëren: De volgende tags komen aan bod:

  • - Header
  • - Nav
  • - Main
  • - Section
  • - Article
  • - Aside
  • - Footer
  • - Div

Uiteraard zijn er nog veel meer tags, maar deze zijn de meest gebruikte.

header

  • Container dat de onderstaande content introduceert.
  • Vaak (maar niet altijd) aan het begin van een nieuwe section.
  • Bevat altijd een H1 en of H2.
  • Je kunt meerdere headers in een website hebben.
  • Footer mag geen header bevatten.

Verschil tussen een head en header

  • head bevat algemente informatie over het document, inclusief de titel en links van scrips en stylesheets
  • header bevindt zich in de body van het document. Bevat kopelementen maar ook logo’s en teksten.

nav

  • Navigatiebar
  • Bevat een lijst/groep met navigatie links.
  • Links om door een website te navigeren.
  • Nav staat vaak bovenaan in de website, maar kan ook aan de zijkant (sidebar)
  • Bevat vak een ul met li, maar ook een titel of een logo.

main

  • Bevat de hoofdinhoud van de website.
  • Main mag maar één keer voorkomen in een document.
  • nav en footer mogen niet in de main.

section

  • Is een “gedeelte” van een document
  • Een website deel je altijd op in "gedeeltes” om het van elkaar te onderscheiden, dit bundel je dan in een section
  • Zit in de main

article

Lijkt op een section, het “bundelt” ook een gedeelte van een document

  • Article gebruik je vaak als er veel tekst staat, denk aan een:
    • - Forum post
    - Blog post
    - Artikel in de krant
  • Er kunnen meerdere sections in een article en meerdere articles in een section
  • Denk aan een bladzijde van een boek. De bladzijde is een section en in de bladzijde heb je een
    • article met de tekst.

aside

  • Staat vaak aan de zijkant van een website,
  • Zie het als een ”side note” .

footer

  • Is het “einde” van een document of section
  • Altijd gebruikt onderaal een document, maar kan ook aan het einde van een section
  • Bevat meestal:
  • - authorship information
  • - copyright informatie
  • - contactinformatie
  • - sitemap
  • - back to top links
  • - related documents
  • Contactinformatie bundel je in een adress tag
  • Als je een footer gebruikt voor het einde van een document, dan altijd buiten de main tag

div

  • Vroeger (HTML4) werd een div overal voor gebruikt. Dit kun je zien bij oude websites als je op inspecteren klikt. Je ziet dat voor alles een div wordt gebruikt.
  • In HTML5 is dit vervangen door alle tags die je vandaag hebt geleerd en meer.
  • Ook een container, maar dan om extra styling toe te voegen of te manipuleren met Javascript.

Tables

Een tabel in HTML bestaat uit tabelcellen in rijen en kolommen. Elke tabelcel wordt gedefinieerd door een td een /td Alles tussen td /td de inhoud van de tabelcel. Elke tabelrij begint met een tr eindigt met een /tr Soms wilt u dat uw cellen tabelkopcellen zijn. Gebruik in die gevallen de th in plaats van de td tag:

Voorbeeld:

<table>
<tr >
<th > Company </ th>
<th > Contact </ th>
<th > Country </ th>
</ tr>
<tr >
<td > Alfreds Futterkiste </td>
<td > Maria Anders </td>
<td > Germany </td>
</ tr>
<tr >
<td > Centro comercial Moctezuma </td>
<td > Francisco Chang </td>
<td > Mexico </td>
</ tr>
</table>

HTML class Attribute:

Het HTML- classattribuut wordt gebruikt om een klasse voor een HTMLelement op te geven. Meerdere HTML-elementen kunnen dezelfde klasse delen. Het classattribuut wordt vaak gebruikt om naar een klassenaam in een stylesheet te verwijzen. Het kan ook door JavaScript worden gebruikt om elementen met de specifieke klassenaam te openen en te manipuleren. Om elementen met een specifieke klasse te selecteren, schrijft u een punt (.) gevolgd door de klassenaam.
In dit voorbeeld zijn alle HTML-elementen met class="center" rood en gecentreerd:

.center {
text-aling: center;
color: red;
}

HTML id Attribute:

Het HTML- idattribuut wordt gebruikt om een uniek ID voor een HTMLelement op te geven. U kunt niet meer dan één element met dezelfde ID in een HTML-document hebben. Het idattribuut wordt gebruikt om naar een specifieke stijldeclaratie in een stijlblad te verwijzen. Het wordt ook door JavaScript gebruikt om het element met de specifieke ID te openen en te manipuleren. Om een element met een specifieke id te selecteren, schrijft u een hekje (#), gevolgd door de id van het element.

Voorbeeld

De onderstaande CSS-regel wordt toegepast op het HTML-element met id="para1":

#para1 {
text-align: center;
color: red;
}

Wat zijn HTML Media Tags?

HTML Media Tags zijn speciale codes waarmee we verschillende soorten media, zoals audio en video, aan onze webpagina's kunnen toevoegen. Stel je voor dat je een coole muziektrack hebt of een geweldige video die je met anderen wilt delen. Met HTML Media Tags wordt dat super makkelijk!

Audio Toevoegen:

Als je een audiofragment hebt dat je wilt laten horen, gebruik dan de audio tag. Door controls toe te voegen, krijg je knoppen zoals afspelen en pauzeren. De 'src' vertelt de browser waar het audiobestand is, en 'type' vertelt het type audiobestand.

Voorbeeld:

<audio controls src="muziek.mp3" type="audio/mp3"> </audio>

Video toevoegen:

Voor video's gebruiken we de video-tag.Net als bij audio kun je met controls video bedienen en kun je de breedte en hoogte instellen met width en height.Om een ​​video automatisch te starten, gebruik je het 'autoplay' attribuut.

<video controls autoplay width="400" height="300" src="movie.mp4" type="video/mp4" ></video>
